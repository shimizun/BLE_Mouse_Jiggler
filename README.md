# 🖱️ Stealthy BLE Mouse Jiggler for Xiao ESP32C6

Xiao ESP32C6を**「Microsoft製のBluetoothマウス」**に擬態させ、PCのスリープや離席状態（離席アイコン）を防止する自動マウスムーバーです。

単に一定間隔で動くのではなく、**「ランダムな待機時間」「滑らかな線形補間移動」「たまに行われる右クリック」**により、より人間らしい挙動をシミュレートします。

---

## ✨ 特徴

- **完璧なステルス性**: デバイス名を `Microsoft Bluetooth Mouse`、製造元を `Microsoft` として登録。
- **人間味のある動き**: マウスカーソルをワープさせず、150ステップに分けて滑らかに移動（Linear Interpolation）。
- **不規則なアルゴリズム**: 
  - 待機時間を10秒〜180秒の間でランダムに決定。
  - 5%の確率で右クリックを実行（メニューを開いて閉じるまでがセット）。
- **省エネ設計**: 接続が切れている間はチェック頻度を下げ、バッテリー残量を80%として通知。

---

## 🛠 準備するもの

### ハードウェア
- **Seeed Studio XIAO ESP32C6** (または他のESP32シリーズ)
- USB-C ケーブル

### ソフトウェア（ライブラリ）
Arduino IDEのライブラリマネージャーから以下のライブラリをインストールしてください：
- [ESP32-BLE-Mouse](https://github.com/T-vK/ESP32-BLE-Mouse) (by T-vK)

---

## 🚀 使い方

1. **環境構築**: Arduino IDEで `ESP32-BLE-Mouse` ライブラリをインストール。
2. **書き込み**: 本リポジトリのスケッチをXIAO ESP32C6に書き込む。
3. **ペアリング**: PCのBluetooth設定を開き、新しいデバイスを追加。一覧に現れる **「Microsoft Bluetooth Mouse」** と接続。
4. **完了**: あとはデバイスを挿しておくだけで、あなたのPCは常に「アクティブ」に保たれます。

## ⚙️ カスタマイズ

コード内の以下の数値を変更することで、お好みの挙動に調整可能です。

```cpp
// ========== 設定項目 ==========
const long minWaitTime = 10000;   // 最小待機時間 (10秒)
const long maxWaitTime = 180000;  // 最大待機時間 (180秒)
const int rightClickChance = 5;   // 右クリックが発生する確率 (%)
// =============================
```
